basic-simplex
=============

[![Build Status](https://travis-ci.org/athanclark/simplex-basic.svg?branch=master)](https://travis-ci.org/athanclark/simplex-basic)

A trivial implementation of primal simplex.

## How to run tests

```
cabal install --enable-tests && cabal test
```
